Common Git Commands

  1. git init :Initializes an empty git repo
  2. git add . : Add files to the staging area
  3. git commit -m "some msg" : Writes changes to the files in staging
  4. git push origin *branch* : Pushing updated changes to remote repo. "main" branch by default
